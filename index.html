Страницы сайта

Просто добавляй новые .html или .htm файлы в эту папку — они сразу станут доступны по адресу /имя-файла

📄 О проекте

about.html — описание сайта и автора

📄 Контакты

➕ Добавить страницу

Создай файл новая-страница.html в этой папке и она появится по адресу /новая-страница.html

Как это работает сейчас

  • Папка открыта через локальный сервер: python -m http.server
  • Все .htm / .html файлы доступны по HTTP (не file://)
  • Скоро подключим бесплатный домен (DigitalPlat / is-a.dev / GitHub Pages)
